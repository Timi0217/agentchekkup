API Reference

POST /api/evaluate
Start async evaluation. Returns eval_id to poll for results.
POST /api/evaluate/sync
Run evaluation synchronously. Blocks until complete.
GET /api/results/{eval_id}
Get results for a specific evaluation run.
GET /api/categories
List all test categories and their test cases.
POST /api/remediate
Find GitHub repos that fix specific test failures. Accepts test_ids or categories.
GET /api/results/{eval_id}/remediation
Get remediation recommendations for a completed evaluation.
GET /api/stats
Get gauntlet usage statistics.
