API Reference

Evaluation
POST /api/evaluate
Start async evaluation. Returns eval_id to poll. Body: {agent_url, protocol, categories?, timeout?, include_remediation?}
POST /api/evaluate/sync
Run evaluation synchronously. Blocks until complete. Returns full results with scorecard, remediation, and prescriptions.
Results
GET /api/results/{eval_id}
Get results for a specific evaluation. Includes scorecard, per-test scores, remediation, deployed fixes, and prescriptions.
GET /api/results
List recent evaluations from persistent history. Query: ?limit=50.
GET /api/results/{eval_id}/fixes
Poll fix deployment status. Returns deployed_fixes array and still_deploying flag for live progress tracking.
Closed-Loop Verification
POST /api/evaluate/{eval_id}/retest
Re-run failed tests with fix prescriptions applied. Wraps agent with pre-input scanning & post-output scrubbing. Returns before/after comparison.
Remediation
GET /api/results/{eval_id}/remediation
Get remediation for a completed evaluation. Includes auto-deployed fixes with prescriptions (hook, API call, system prompt patch).
POST /api/remediate
Find GitHub repos that fix specific failures. Body: {test_ids?, categories?}. Returns deploy-ready repos with Chekk deploy commands.
Discovery
GET /api/categories
List all 6 test categories with test counts and severity levels.
POST /api/tests
List all 29 test cases with descriptions and check types. Body: {categories?} to filter.
GET /api/stats
Service stats: total evaluations, running count, completed count, agent calls.
